# 学习编程有没有必要做笔记? 如果做笔记?
1. 学编程有没有必要做笔记?
    + A: 十分肯定, 一定要做!
2. 我该记哪些内容?
    + A: 记笔记有 `3 记` 和 `3 不记`
        - **3 记**:
            + (1) 涉及到软件安装和配置 记
            + (2) 涉及到项目实战技能 记
            + (3) 涉及到出错问题必须记
        - **3 不记**:
            + (1) 书上有的知识点不记 (抛去上面 3 记)
            + (2) 网上已有完善的参考手册不记
            + (3) 看书的时候一定不记.


## 计算机的体系与结构

- 冯诺依曼体系和结构
    * 冯诺依曼体系:
        + (1) 能够把需要的程序和数据送至计算机中

        + (3) 能够具备算术、逻辑运算和数据传送等数据加工处理的能力
